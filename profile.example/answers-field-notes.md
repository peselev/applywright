# Application Answers — Field Notes

Human-maintained, with agent proposals. The agent only writes here after showing
a proposal and getting your approval (see `skills/shared/rating-and-learning.md`).

Purpose: track what kinds of application-form answers land, by question type, so
future drafts start from proven framings. The `application-answers` skill reads
this file before drafting.

---

## How to use this file

Entries accumulate from **starred** (⭐) answers. Each entry names a question type,
the framing that worked, and what the skill should do by default next time.

Entry format:

```
### {question type — e.g. "why this company", "describe a failure"}
Framing that worked: {the transferable move}
Default going forward: {what the skill should reach for by default}
Source: {short-id} ({date})
```
