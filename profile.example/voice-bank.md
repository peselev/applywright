# Voice Bank

Your own writing, kept verbatim. When you hand the agent a substantial piece of
your own prose — a draft you wrote, an old letter or answer you're sharing, a
paragraph you typed to steer a piece — it confirms the text is yours and saves it
here exactly as written. No analysis, no summary, no rewrite.

The agent reads this file when it drafts something new (see
`skills/shared/drafting-protocol.md`): it's where your real voice, phrasing,
recurring facts, and original ideas come from. The richer this bank, the less the
agent has to ask before it can write something that sounds like you. On a fresh
system it's empty, so early pieces will prompt you to seed it — that's expected.

The agent only appends here after you confirm a passage is your own writing, and
it never edits what's banked. You own this file.
